## Usage

The solidity and yul contracts can be built using hardhat:

```
npm install
npx hardhat compile
```

The test suite uses a hardhat fork of the Ethereum mainnet. After specifing an archive node in `.env`, tests can be run with:

```
npx hardhat test
```
